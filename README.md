# AdGuardFilters

AdGuard 内容过滤规则。

## 目录

```text
|--AdGuardFilters
   |--duplicate
   \--my
```

```text
|--AdGuardFilters
   |--duplicate  从其他现成的仓库、博客复制过来的规则，会注明来源及其主页
   \--my         我自己在日常使用AdGuard过程中积累起来的规则，可能会有不普遍适用、误拦截的情况
```

## 注

规则语法请查阅官方文档： [How to create your own ad filters](https://kb.adguard.com/zh/general/how-to-create-your-own-ad-filters) 。
