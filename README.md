# AdGuardFilters

AdGuard 内容过滤规则。

```text
|--AdGuardFilters
   |--duplicate
   |  |--android
   |  \--pc
   \--my
      |--android
      \--pc
```

```text
|--AdGuardFilters
   |--duplicate  从其他现成的仓库、博客复制过来的规则，会注明来源及其主页
   |  |--android   特指只能用于安卓的规则
   |  \--pc        特指只能用于PC的规则
   \--my         我自己在日常使用AdGuard过程中积累起来的规则，可能会有不普遍适用、误拦截的情况
      |--android   特指只能用于安卓的规则
      \--pc        特指只能用于PC的规则
```
